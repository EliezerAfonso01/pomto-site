📋 Perguntas Gerais

O POMTO (Pombo Remoto) é uma solução completa para trabalho remoto que combina organização de tarefas, monitoramento ético da produtividade e cuidados com a saúde digital.

Nossa missão é transformar o trabalho remoto em uma experiência mais saudável, produtiva e respeitosa, tanto para trabalhadores quanto para empresas.

Diferencial: Focamos em resultados e bem-estar, não em vigilância invasiva.

Sim! O POMTO oferece uma versão completamente gratuita para uso individual que inclui:

  • Organização pessoal de tarefas
  • Lembretes de pausas saudáveis
  • Frases motivacionais
  • Dashboard básica de produtividade
  • Participação em múltiplas empresas

Também temos planos pagos com funcionalidades avançadas como CV automático e dashboards empresariais.

Absolutamente! A privacidade é um dos nossos pilares fundamentais. O POMTO foi desenvolvido com base em princípios éticos:

  • Não capturamos telas nem conteúdo pessoal
  • Não acessamos arquivos privados
  • Focamos em métricas de produtividade, não vigilância
  • Você controla quais dados compartilhar
  • Todos os dados são criptografados
Importante: Leia nossa Política de Privacidade para entender completamente como seus dados são tratados.

💻 Instalação e Configuração

Atualmente, o POMTO está disponível para:

  • Windows 10/11: Versão completa via Microsoft Store
  • Web: Dashboard empresarial via navegador

Em breve, o POMTO estará disponível para macOS, Linux e dispositivos móveis. Acompanhe nosso blog e junte-se à comunidade no WhatsApp para atualizações em primeira mão.

Link direto: Microsoft Store

A configuração inicial é simples e leva apenas alguns minutos:

  1. Baixe e instale o POMTO da Microsoft Store
  2. Crie sua conta com email e senha
  3. Configure seus horários de trabalho
  4. Defina suas tarefas
  5. Personalize os lembretes de saúde
  6. Comece a usar!

Ainda não temos um assistente automático de configuração, mas temos vídeos no nosso canal no YouTube que mostram passo a passo como preparar o ambiente.

Requisitos mínimos para Windows:

  • Windows 10 versão 1809 ou superior
  • 2GB de RAM
  • 200MB de espaço em disco
  • Conexão com internet (opcional, para recursos online)

Recomendado:

  • Windows 11
  • 4GB de RAM ou mais
  • Conexão banda larga estável (para melhor experiência)

O aplicativo pode funcionar sem internet, mas alguns recursos dependem de conexão para oferecer a melhor experiência.

⚙️ Funcionalidades

Os relatórios do POMTO são gerados com base em métricas éticas de produtividade:

  • Tempo ativo: Períodos de atividade no computador
  • Tarefas concluídas: Objetivos marcados como finalizados
  • Pausas realizadas: Cumprimento dos intervalos de saúde (brevemente)
  • Aplicações utilizadas: Categorização de ferramentas de trabalho

Todos os dados podem ser exportados em PDF, Csv e em Excel.

Importante: Não monitoramos conteúdo, apenas padrões de atividade.

O POMTO cuida da sua saúde digital com lembretes inteligentes:

  • Pausas regulares: Intervalos de 5-10 minutos a cada hora
  • Exercícios para os olhos: Dicas para descansar a visão
  • Alongamentos: Exercícios simples para o corpo
  • Hidratação: Lembretes para beber água
  • Postura: Dicas de ergonomia

Você pode personalizar a frequência e o tipo de lembretes conforme suas necessidades.

O sistema de organização do POMTO é baseado em objetivos SMART:

  • Específicos: Defina tarefas claras e detalhadas
  • Mensuráveis: Estabeleça critérios de conclusão
  • Alcançáveis: Metas realistas para seu contexto
  • Relevantes: Alinhadas com seus objetivos profissionais
  • Temporais: Com prazos definidos

Você pode criar projetos, dividir em subtarefas e acompanhar o progresso em tempo real.

Sim! O POMTO foi desenvolvido pensando em freelancers e profissionais que trabalham para múltiplas empresas:

  • Perfis separados para cada empresa
  • Relatórios individualizados
  • Controle de tempo por projeto
  • Dashboard unificada para você
Vantagem: Uma única instalação para gerenciar todos os seus trabalhos.
